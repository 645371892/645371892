热设计中的风扇

发表于 2019-05-26 | 阅读数
字数统计: 1k | 阅读时长 ≈ 3

风扇类型

电子设备中，风扇是用来加速空气流动的部件，可以提高对流换热系数。又称为风机，分为轴流风扇和离心风扇。

轴流风扇

轴流风扇：风扇的进风口与出风口平行。

风量大、风压小、噪音小，种类繁多、价格便宜。

离心风扇

离心风扇：风扇的进风口与出风口垂直。

风量小、风压高、噪音大，价格相对较高。

风扇的内部构造包括外框，PCB，轴承，扇叶等。其中轴承和叶片是风扇可靠性的关键决定因素。

技术指标

风量

风量是指风扇每分钟排出或纳入的空气总体积，常用单位为CFM（Cubic Feet per Minute，立方英尺/分钟）。风量是衡量风扇散热能力的最重要的指标，风量越大的散热风扇其散热能力也越高。

风压

风压，又称为静压（Static Pressure），是指风扇在进气端与出气端的气压差异，常用单位为公厘水柱高度（mmH2O）或是英寸水柱高（inchH2O）。一般情况下，风压越大越好，表示风扇可以为机箱内部更深的空间提供散热。风压与风量的关系是相对的，在固定转速下，风扇的风压越大，则风量越小，反之亦然。如果想同时提高风扇的风量和风压，除了扇叶设计之外，可以提高风扇转速。

各风扇厂家提供的产品规格书中，风量与风压的单位可能不同，可参考下表进行换算。

噪音

风扇噪音是风扇工作时产生杂音的大小，受多方面因素影响，单位为分贝(dB)。

风扇噪音与摩擦力、空气流动有关。风扇转速越高、风量越大，造成的噪音也会越大。在风量相同的情况下，大风扇在较低转速时的工作噪声要小于小风扇在高转速时的工作噪声。

风扇选型

风扇选型时，需要关注的核心参数为尺寸、风量、风压、噪音、寿命、成本。风扇的选型是产品热设计方案的核心所在，需要综合考虑前述所有温度、噪音、几何空间、成本、运行寿命等所有方面的要求。值得指出，风扇的PQ特性曲线和风扇的工作点，在散热分析中有重要价值。

P-Q曲线

对于风扇的风压风量规格与特性测试，主要是依循美国Air Movement and Control Association (AMCA) 编号 210-85之“Laboratory Method of Testing Fans for Rating”测试规范进行。而其量得的特性曲线，即是一般通称的P-Q曲线。

P-Q 曲线，一般常以3个物理值来描述它：

  • Pmax：当风量为 0 时，在某密闭空间的固定容积状态下，该风扇的最大静压值；
  • Qmax：当风扇入口与出口二端压力差为 0 时的流量，即为该风扇的最大流量；
  • PQ值：为风扇入口与出口二端压力差与当时状态下流量Q的对应值；

风扇的P-Q曲线和工作点位置示意

1.工作点位于右侧1/3区域，证明系统风阻较低，可以尝试：

  • 加密散热齿，增加换热面积；
  • 更换风压低但风量高的风扇；

2.工作点位于中间的不稳定区域，有可能会出现失速，应当尽力避免；

3.工作点位于左侧1/3，即较靠上的位置时，证明系统阻力较大，可以尝试：

  • 加大进出风口；
  • 减少流动路径上的结构件；
  • 稀疏化散热齿；
  • 更换风压更大的风扇。

风扇的选型在强迫风冷设计中是核心工作，对产品的最终散热表现、噪音表现和系统结构设计方案都有关键影响。
